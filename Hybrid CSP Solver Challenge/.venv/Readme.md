# Project Setup

**Dataset:** [ZebraLogicBench](https://huggingface.co/datasets/allenai/ZebraLogicBench)

This project uses Python 3.14. All scripts are located in `src/main/`.

## Installation

To install all necessary libraries, run:

```bash
  pip install -r requirements.txt
```

# Dataset Import
````bash
  python src/setup/download_dataset.py

````

# Dataset Test
````bash
  python src/examples_and_tests/load_dataset_test.py

````

# Mini guide:
Dataset Configurations Mini-Guide

grid_mode → Dataset in a tabular, grid-like format.

tree_mode / mc_mode → Different internal representations for testing or modeling purposes.

Tree / MC are typically used for specific tasks, e.g., logical decision trees or Monte-Carlo simulations.

Which mode to choose depends on how you want to analyze or train on the data.

For simple analysis or ML training, grid_mode is usually the easiest and most straightforward to work with.
